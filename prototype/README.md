# Prototype Inbox

Place the current website prototype for import here.

Recommended options:
- raw HTML / CSS / JS files
- exported design prototype files
- screenshots plus a reference HTML file

Suggested naming:
- `index.html`
- `styles.css`
- `app.js`
- `reference.png`

Once the prototype is placed here, the next step is:
- inspect the prototype
- preserve what is strong
- refactor it into the structured portfolio site module
- make it editable through the shared site content model
