# Site Brief

## Core Mission

Create a portfolio website for Valentin Rodin that feels:
- cinematic
- intentional
- premium
- international
- casting-friendly

This site should not read like a generic actor template.
It should feel like a serious screen presence presentation for a mature male actor working across Asia.

The site should follow the structure of the portfolio PDF first,
then improve it for the web.

## Positioning

Primary positioning:
- international actor working across Asia

Secondary positioning:
- model
- on-camera talent
- spokesperson

The site should support:
- casting directors
- agencies
- production companies
- brand / commercial clients

## Priority Materials

- IMDb
- Intro Video
- Portfolio PDF
- YouTube channel / selected episodes
- contact details
- Instagram

## Non-Negotiable Website Blocks

- profile / physical facts
- key acting credits
- presentation video window
- gallery / lookbook
- training / special skills
- featured YouTube block
- materials + contact

## Key Archetypes To Present

The site should help communicate the following visual / character directions:

1. Detective
2. Romantic
3. Business / James Bond
4. Lover
5. Macho
6. Sportsman

These do not need to become cliche costume categories.
They should read as believable casting energies / screen archetypes.

## Design Intent

- bold but clean
- cinematic rather than corporate
- readable on mobile
- strong hero section
- visible calls to action
- easy path to materials

Avoid:
- generic template feel
- overstuffed one-pager chaos
- weak typography
- bland corporate colors
- portfolio PDF vibe copied directly into the site

## Early Site Questions To Resolve In Build

1. Should the site feel more actor-first or actor + commercial hybrid?
2. How many archetype blocks should be visible on the homepage versus deeper sections?
3. Should the intro video appear in the hero or immediately below it?
4. What is the strongest first impression:
   - cinematic portrait
   - archetype grid
   - short positioning statement
   - reel-first landing

## Recommended First Build Order

1. Homepage structure based on the PDF portfolio
2. Hero section
3. Presentation video section directly below hero
4. Credits / profile structure
5. Archetype / range section
6. Gallery section
7. YouTube section
8. Materials section
9. Contact section
10. Mobile pass
