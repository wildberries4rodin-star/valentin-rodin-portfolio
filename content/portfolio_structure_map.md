## Core Rule

The website should follow the structure of the portfolio PDF,
but it should not become a literal page-by-page web copy.

Meaning:
- the PDF defines what must be represented
- the website defines how it should be experienced

## Website Structure To Mirror The Portfolio

1. Hero
- strong first image
- clear positioning
- immediate CTA to intro video and portfolio PDF

2. Profile / Physical Details
- name
- age / DOB
- nationality
- languages
- height / weight
- sizing / shoes
- hair / eyes
- tattoo / notable visual details
- contact handles

3. Selected Acting Credits
- SBS drama
- The Wandering Earth
- Commando 3
- The Next Prince
- any additional strongest credits that appear in the portfolio

4. Presentation Video
- embedded video window
- visible high on the page
- this is not optional

5. Character Range / Archetypes
- detective
- romantic
- business / Bond
- lover
- macho
- sportsman

6. Gallery / Lookbook
- strongest portraits
- acting stills
- selected commercial/editorial images

7. Training / Special Skills
- acting training
- hosting
- physical skills
- language ability

8. YouTube / Content Creator Block
- featured episodes
- direct channel link
- positioned as supporting proof of personality and on-camera confidence

9. Materials
- IMDb
- PDF portfolio
- intro video
- social links

10. Contact
- KakaoTalk
- Instagram
- booking / inquiry logic

## What The Site Adds Beyond The PDF

- better hierarchy
- easier casting scan
- embedded media
- mobile readability
- cleaner updates through the agent

The site architecture should now be built around this structure.
